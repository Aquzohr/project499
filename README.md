# Web Application for Powerhouse fittness Center

This is Project for 499 at CS CMU

## Getting Started

My project build with Ruby on Rails Version 5.0+

Database: Postgresql

### Installing

Used followed comand

```
 $ bundler
 $ rails db:setup
 $ rails s
```
